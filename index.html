Innovation & Tooling
RAP Code Generator — AI-Powered Developer Tool
Built from scratch and adopted across the organization for accelerating ABAP RAP development
+

What I Built

A web-based tool hosted on GitHub Pages with a Cloudflare Worker backend that uses AI (Claude API) to auto-generate CDS views, Behavior Definitions, Handler Classes, and Projections from natural-language prompts.

Impact

Adopted by developers across the organization. Significantly reduces boilerplate coding time for RAP objects, enabling the team to focus on business logic rather than repetitive scaffolding.

Key Features Delivered

SSE streaming with progressive tab rendering, token exhaustion handling, guided "Take a Tour" onboarding, "Report Issue" workflow (FormSubmit.co), and upgrade to Claude Sonnet 4 with expanded token limits.

Ownership

End-to-end — ideation, architecture, frontend, backend proxy, API integration, UX iterations, internal promotion, and ongoing maintenance. This was a personal initiative that grew into an org-level asset.

RAP Application Development
Intra-Company Asset Transfer Application
Full-lifecycle RAP solution with Excel upload, BAPI integration, approval workflows, and event-driven notifications
+

What I Built

A RAP-based application enabling intra-company asset transfers with Excel file upload, BAPI integration (BAPI_ASSET_TRANSFER_POST, BAPI_FIXEDASSET_CHANGE), multi-level approval workflows, and background processing via Application Jobs.

Impact

Streamlined a previously manual and error-prone process into a governed, auditable digital workflow with real-time notifications and status tracking, reducing processing time and eliminating data inconsistencies.

Integration Layer

Connected approval events with SAP Event Mesh for real-time notifications. Implemented RAP Entity Events for loose coupling. Built mass asset acquisition (ABSOL) upload tool as an extension.

Technical Depth

Comprehensive validation framework using XCO library for Excel parsing, CDS-based authorization with PFCG integration, and Application Jobs for asynchronous processing of large transfer batches.

SAP Event Mesh & CPI Integration
Designed and implemented event-driven communication between SAP BTP applications and downstream systems
+

What I Delivered

Set up SAP Event Mesh integration with outbound bindings and topic configuration for real-time event propagation. Configured RAP Entity Events to publish business events upon approval actions in custom applications.

Impact

Enabled real-time, event-driven workflows replacing batch-based polling, resulting in faster notification delivery and reduced system coupling between BTP apps and on-premise systems.

Architecture Contribution
