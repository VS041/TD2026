RAP Application Development
Asset Transfer RAP Application — Core Build
Full-lifecycle RAP solution with Excel upload, BAPI integration, approval workflows, and event-driven notifications
+

What I Built

A RAP-based application enabling intra-company asset transfers with Excel file upload, BAPI integration (BAPI_ASSET_TRANSFER_POST, BAPI_FIXEDASSET_CHANGE), multi-level approval workflows, and background processing via Application Jobs.

Impact

Streamlined a previously manual and error-prone process into a governed, auditable digital workflow with real-time notifications and status tracking, reducing processing time and eliminating data inconsistencies.

Integration Layer

Connected approval events with SAP Event Mesh for real-time notifications. Implemented RAP Entity Events for loose coupling. Built mass asset acquisition (ABSOL) upload tool as an extension.

Technical Depth

Comprehensive validation framework using XCO library for Excel parsing, CDS-based authorization with PFCG integration, and Application Jobs for asynchronous processing of large transfer batches.

Integration & Event-Driven Architecture
Event Mesh Integration & Approval Workflows
Designed and implemented event-driven communication between SAP BTP applications and downstream systems
+

What I Delivered

Set up SAP Event Mesh integration with outbound bindings and topic configuration for real-time event propagation. Configured RAP Entity Events to publish business events upon approval actions in custom applications.

Impact

Enabled real-time, event-driven workflows replacing batch-based polling, resulting in faster notification delivery and reduced system coupling between BTP apps and on-premise systems.

Debugging & Problem Solving

Diagnosed and resolved CPI connectivity issues with Event Mesh — traced problems through message queues, topic subscriptions, and integration flow configurations to ensure end-to-end message delivery.

Architecture Contribution

Established reusable patterns for event-driven communication in RAP-based applications. Documented outbound binding setup, topic naming conventions, and error-handling strategies for the team.

Innovation & Tooling
RAP Generator — Personal Innovation Initiative
Built from scratch: an AI-powered code generator for ABAP RAP development, hosted on GitHub Pages with Cloudflare Workers backend
+

What I Built

A web-based tool hosted on GitHub Pages with a Cloudflare Worker backend that uses AI (Claude API) to auto-generate CDS views, Behavior Definitions, Handler Classes, and Projections from natural-language prompts.

Impact

Significantly reduces boilerplate coding time for RAP objects, enabling developers to focus on business logic rather than repetitive scaffolding. Demonstrated a practical intersection of AI and SAP development.

Architecture

Frontend on GitHub Pages (vanilla JS), Cloudflare Worker as backend proxy for Claude API calls, SSE streaming for real-time code generation output. Zero-cost hosting with edge-deployed backend.

Ownership

End-to-end personal initiative — ideation, architecture, frontend, backend proxy, API integration, and UX iterations. Built entirely outside of regular project work as a self-driven innovation effort.

ABAPLens & Next-Gen SAP Developer Tools
A suite of AI-powered micro-SaaS tools for SAP developers — Reverse Spec Generator, Visual Flow Mapper, Impact Blast Radius, and more
+

What I'm Building

ABAPLens — a suite of AI-powered tools for SAP developers featuring Reverse Spec Generator (code → functional spec), Visual Flow Mapper (ABAP logic → visual flowcharts), Impact Blast Radius (change impact analysis), and Test Case Generator.

Vision & Impact

Addressing real pain points in SAP development: undocumented legacy code, lack of visual debugging tools, time-consuming impact analysis, and manual test case creation. Each tool saves hours of developer effort per use.

Broader Tool Ideas

Also exploring Error Decoder (SAP error → plain English fix), Functional Spec to Technical Spec Converter, and ABAP Code Explainer — building toward a comprehensive SAP developer productivity platform.

Tech Stack & Approach

Cloudflare Workers + GitHub Pages stack (proven with RAP Generator). Claude API for AI capabilities. Designed as standalone micro-SaaS tools that can be used independently or as a suite.
